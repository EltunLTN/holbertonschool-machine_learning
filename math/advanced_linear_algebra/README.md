# Math - Advanced Linear Algebra

This directory contains advanced linear algebra tasks from the Holberton machine learning curriculum.
It focuses on determinant-based matrix operations, cofactors, adjugates, inverses, and matrix definiteness.

## Learning Objectives

- Compute determinants recursively
- Build minor and cofactor matrices
- Compute the adjugate of a square matrix
- Find the inverse of a matrix when it exists
- Determine whether a matrix is positive or negative definite

## Files

- `0-determinant.py`: Defines `determinant(matrix)`
- `1-minor.py`: Defines `minor(matrix)`
- `2-cofactor.py`: Defines `cofactor(matrix)`
- `3-adjugate.py`: Defines `adjugate(matrix)`
- `4-inverse.py`: Defines `inverse(matrix)`
- `5-definiteness.py`: Defines `definiteness(matrix)`

## Requirements

- Python 3.x
- `pycodestyle` style compliance (where required by checker)
- Executable scripts with shebang where required

## Usage

Run Python task files with:

```bash
python3 <filename.py>
```

Example:

```bash
python3 0-determinant.py
```
