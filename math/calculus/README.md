# Math - Calculus

This directory contains introductory calculus tasks from the Holberton machine learning curriculum.
It includes short theory answer files and Python functions for polynomial derivative/integral and series summation.

## Learning Objectives

- Understand common calculus notation (sigma, pi, derivatives, integrals)
- Connect discrete sums with closed-form expressions
- Implement basic calculus-related functions in Python
- Validate function inputs and return expected edge-case values

## Requirements

- Python 3.x
- `pycodestyle` style compliance (where required by checker)
- Executable scripts with shebang where required

## Usage

Run Python task files with:

```bash
python3 <filename.py>
```

Example:

```bash
python3 9-sum_total.py
```
