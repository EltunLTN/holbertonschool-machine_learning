# Math - Linear Algebra

This directory contains introductory linear algebra tasks from the Holberton machine learning curriculum.
It covers list-based matrix operations, basic NumPy usage, slicing, transposition,
matrix multiplication, and matrix operations on higher dimensions.

## Learning Objectives

- Represent vectors, arrays, and matrices with Python lists and NumPy arrays
- Compute matrix shape and transpose matrices
- Perform element-wise operations and matrix multiplication
- Concatenate arrays/matrices across different axes
- Apply matrix slicing in plain Python and NumPy
- Handle invalid shapes and dimensions with safe return values

## Files

- `0-slice_me_up.py`: Slice a Python list and print specific sections
- `1-trim_me_down.py`: Slice a 2D list and print middle columns
- `2-size_me_please.py`: Defines `matrix_shape(matrix)`
- `3-flip_me_over.py`: Defines `matrix_transpose(matrix)`
- `4-line_up.py`: Defines `add_arrays(arr1, arr2)`
- `5-across_the_planes.py`: Defines `add_matrices2D(mat1, mat2)`
- `6-howdy_partner.py`: Defines `cat_arrays(arr1, arr2)`
- `7-gettin_cozy.py`: Defines `cat_matrices2D(mat1, mat2, axis=0)`
- `8-ridin_bareback.py`: Defines `mat_mul(mat1, mat2)`
- `9-let_the_butcher_slice_it.py`: NumPy slicing on a 2D array
- `10-ill_use_my_scale.py`: Defines `np_shape(matrix)`
- `11-the_western_exchange.py`: Defines `np_transpose(matrix)`
- `12-bracin_the_elements.py`: Defines `np_elementwise(mat1, mat2)`
- `13-cats_got_your_tongue.py`: Defines `np_cat(mat1, mat2, axis=0)`
- `14-saddle_up.py`: Defines `np_matmul(mat1, mat2)`
- `100-slice_like_a_ninja.py`: Defines `np_slice(matrix, axes={})`
- `101-the_whole_barn.py`: Defines `add_matrices(mat1, mat2)` for any dimension
- `102-squashed_like_sardines.py`: Defines `cat_matrices(mat1, mat2, axis=0)` for any dimension

## Requirements

- Python 3.x
- NumPy (for NumPy-based tasks)
- `pycodestyle` style compliance (where required by checker)
- Executable scripts with shebang where required
