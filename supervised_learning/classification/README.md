# Deep Neural Network

This directory contains the implementation of a deep neural network for binary classification.

## File

- 16-deep_neural_network.py: defines the DeepNeuralNetwork class with validated constructor inputs, private attributes for layers, cache, and weights, and He initialization for all network parameters.

## Features

- Supports any number of hidden layers.
- Stores intermediate activations in a cache dictionary.
- Exposes read-only properties for the number of layers, cached values, and weights.
